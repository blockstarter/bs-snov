# Blockstarter SNOV



### Setup 

1. Please updase update node (the version should be higher than v1)

```
npm i n -g 
n latest
```

2. Please check `sass -v` or install it http://sass-lang.com/install

3. Run
```sh



npm run setup

```

### If Blockstarter asks for update 

```sh 

npm run update

```

## Compile 

```sh 

npm run compile

```

## Run Server (Debug)

```sh
npm run start 
```

## Run Server

```sh

#npm run forever

# --> http://localhost:8080

```

## Stop Server 

```
npm run stop
```
